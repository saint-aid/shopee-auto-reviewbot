# Shopee 리뷰 자동 응답 MVP
*shopee-review-autobot*


Shopee 셀러를 위한 리뷰 자동 응답 시스템입니다.

## 설치 방법
1. 저장소 클론
2. 필요한 패키지 설치: `pip install -r requirements.txt`

## 사용 방법
1. `streamlit run app.py`
2. Shop ID 입력
3. 로그인 진행
4. 리뷰 수집 및 응답 생성
5. 리뷰 자동 등록
